# this

## 概念

**当前执行上下文**的**一个属性**，在非严格模式下，总是指向一个对象，在严格模式下可以是任意值。

## this 指向总归四种情况

#### 1.默认绑定

> 直接使用而不带任何修饰的函数调用 ，就 默认且只能 应用 默认绑定。

#### 2.隐性绑定

> 函数里的 this 默认绑定为上下文对象

#### 3.显性绑定

> call ，apply，bind

#### 4.new

## 其他

**优先级**
new 绑定 > 显示绑定 > 隐式绑定 > 默认绑定

**备注**

- 箭头函数的 this 绑定无法被修改；
- this 默认绑定到哪呢，一般是`window`上，严格模式下 是`undefined`；
